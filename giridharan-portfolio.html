AI-Driven Student Data Management & Power BI Dashboard

Project Pipeline – AI-Driven Student Analytics & Business Intelligence
① Ingestion
Student registration and attendance data are captured through a web interface and stored in a normalized MySQL database.
② Transformation
Automated SQL triggers validate incoming records and generate derived business data, while analytical SQL views transform transactional data into reporting-ready datasets.
③ Modelling
Power BI connects directly to the MySQL database, where Power Query performs data extraction of the data model. A star schema is implemented with attendance as the fact table and students, courses, and date as dimension tables to optimize analytical performance.
④ Insights
Business metrics are created using DAX to analyze registrations, attendance, course performance, instructor efficiency, fee collection, student demographics, and at-risk learners. The final output consists of interactive dashboards that convert raw operational data into actionable insights for academic and business decision-making.
🤖
Built an AI-enabled web application to automate student data retrieval, cutting manual record-handling effort by an estimated 40% and reducing lookup time from minutes to seconds.

Project Pipeline – Grocery Store Sales Analytics & Business Intelligence
① Collection
Daily grocery sales data maintained in Excel is collected, validated, and loaded into a normalized MySQL database, where relational modeling, constraints, and customer deduplication ensure accurate, consistent, and analysis-ready data.

④ Dashboards
The processed data is visualized through interactive dashboards that provide actionable insights into sales performance, customer behavior, and inventory demand, enabling data-driven decision-making for retail operations.
